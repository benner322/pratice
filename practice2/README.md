# Визуализатор зависимостей Python пакетов

Инструмент для анализа и визуализации графа зависимостей пакетов Python.

## Установка зависимостей
py -m pip install requests graphviz

## Параметры командной строки
-pn NAME - Имя пакета (обязательно)
-at 1 - Показать ASCII-дерево
-o NAME - Имя выходного файла
-f TEXT - Фильтр зависимостей
-d N - Глубина анализа
-u URL - URL репозитория
-t 0/1 - Тестовый режим
## Основные команды
Запуск:
py practice.py -pn "имя пакета"
Например: py practice.py -pn requests
Запуск с ошибкой:
py practice.py -pn django -d "zxc"
Запуск с ASCII-деревом:
py practice.py -pn django -at 1

## Результаты
- SVG файл - визуализация графа зависимостей
- ASCII-дерево - текстовое представление в консоли
- Список зависимостей - прямые зависимости пакета
